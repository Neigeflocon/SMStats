# Cvars

```
xstats_enabled
  - Should the tracking plugin be enabled?. (Default: 1)

xstats_debug
  - Debug. (For development purposes) (Default: 0)

xstats_allow_bots
  - Should bots be allowed to be tracked as a valid opponent?. (Default: 0)

xstats_allow_warmup
  - Should warmup be a valid round to track?. (Default: 0)

xstats_serverid
  - Server ID the server should be identified as. (Must be unique and match database server id) (Default: 1)

xstats_minimumplayers
  - Minimum amount of players required. (Default: 4)
```
